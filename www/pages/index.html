About this server

What is webserv?

A fully functional HTTP/1.0 server in C++98 — think miniature NGINX, built with raw syscalls, zero mercy for blocking I/O, and a single event loop driving everything.

webserv is the final systems project at 42. It handles real HTTP requests from real browsers — parsing requests, routing them, executing CGI, serving static files, and returning accurate status codes and headers.

The entire server is non-blocking. A single poll() call drives all I/O across every client simultaneously. It listens on multiple ports and survives stress testing without a single crash.

Built strictly to C++98 with no external libraries — every feature from the HTTP parser to CGI integration is hand-crafted from scratch.
